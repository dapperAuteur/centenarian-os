---

## Course Description

This course teaches you the full Planner module: how to build a multi-decade roadmap, break it down into actionable goals and milestones, schedule tasks in the daily planner, automate recurring work, track financial costs and revenue, manage your planning history with archiving, and use the AI-powered weekly review to stay on track.

The first two lessons are sequential (the hierarchy and how to build one). After that, the course branches so you can explore any feature in any order.

## CYOA Structure

**Linear opening (lessons 01–02):** All users complete Welcome and The Goal Hierarchy in sequence. The hierarchy is the mental model for everything else.

**Topic menu crossroads (after lesson 02):** Users choose based on what they want to do next.

Recommended paths:
- *Just want to get set up?* → 03 → 04 → 05 → 06 → 07
- *Already have a roadmap, want to use the planner daily?* → 07 → 08
- *Want to track project finances?* → 09
- *Want to see the AI review in action?* → 11
- *Need to clean up old plans?* → 10
- *Want to link tasks to contacts, locations, or other modules?* → 12 → 13
