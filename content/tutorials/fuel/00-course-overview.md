## Course Description

This course teaches you everything in the Fuel & Nutrition module: how to build an ingredient library, write personal meal protocols, track daily meals, manage inventory, log batch cooking sessions, and use AI-powered recipe suggestions. The course is built around the NCV (Nutrient vs. Calorie Value) framework — the scoring system at the heart of how CentenarianOS evaluates food quality.

After the first two foundational lessons, the course branches into a free-choice topic menu. You can go deep on any feature in any order.

## CYOA Structure

**Linear opening (lessons 01–02):** All users complete Welcome and The NCV Framework in sequence. These are foundational — the rest of the course makes more sense with this context.

**Topic menu crossroads (after lesson 02):** Users see a branching menu to choose their next topic. Recommended starting points based on use case:

- *Just want to log meals?* → Start with 03 (Ingredients) → 04 (Protocols) → 05 (Meals)
- *Batch cook and track inventory?* → 03 → 07 (Inventory) → 08 (Meal Prep)
- *Explore AI features?* → 10 (AI Recipe Ideas)
- *Share recipes publicly?* → 04 → 11 (Publishing)
