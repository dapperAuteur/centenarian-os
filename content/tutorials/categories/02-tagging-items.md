## Narrator Script

Now that you know what Life Categories are, let's tag some items. Tagging is fast — a couple of clicks from any edit modal. This lesson shows you exactly where to find the tagger and how to add or remove categories.

---

### Finding the Tagger

The Life Categories tagger appears automatically in every entity edit modal. Look for the **"Life Categories"** section with a tag icon below the Activity Links section:

- **Tasks** — open any task in the Planner, click to edit
- **Workouts** — click the link/detail icon on any workout log
- **Invoices** — click the link icon on any invoice row
- **Trips** — click the link icon on any trip
- **Fuel Logs** — click the link icon on any fuel entry
- **Maintenance** — click the link icon on any service record
- **Recipes** — click the link icon on any recipe
- **Equipment** — scroll down on any equipment detail page
- **Focus Sessions** — open the edit modal for any session

---

### Adding a Tag

1. Find the **Life Categories** section in the edit modal
2. Click the **+ Tag** button with the dropdown arrow
3. The dropdown shows all available categories (ones not already assigned)
4. Click a category — the tag appears instantly as a colored chip

The tag is saved immediately. No need to click a separate save button.

---

### Removing a Tag

Each tag chip shows a small **X** button on the right side. Click it to remove the tag. Like adding, removal is instant.

---

### Multiple Tags

Items can have as many life category tags as you want. A grocery trip could be tagged:
- **Health** (buying healthy food)
- **Home** (household supplies)
- **Finance** (part of your grocery budget)

Each tag shows as a separate colored chip with the category's color.

---

### Compact Mode (Uncategorized Items View)

On the Categories dashboard's uncategorized items list, you'll see a compact version of the tagger:

- Tags appear as **small colored dots** instead of full chips
- A dashed **+** circle lets you add a tag
- Hover over a dot to see the category name
- Click a dot to remove it

This compact mode lets you tag many items quickly without the full modal.

---

### Managing Categories

If the 8 default categories don't fit your needs, you can customize them from the Categories dashboard:

1. Go to **Dashboard > Life > Categories** (`/dashboard/categories`)
2. Click the **Manage** button in the header
3. From here you can:
   - **Rename** a category — click the name to edit
   - **Recolor** — pick a new color from the palette
   - **Reorder** — change the sort order
   - **Delete** — removes the category and all its tags (items return to uncategorized)
   - **Add new** — create a custom category with any name and color

## Key Takeaways

- The Life Categories tagger appears in every entity edit modal, below Activity Links
- Click "+ Tag" to add, click X to remove — changes save instantly
- Items can have multiple life category tags
- Compact mode (colored dots) is used in the uncategorized items view for quick tagging
- Manage categories from the Categories dashboard — rename, recolor, reorder, delete, or create new
