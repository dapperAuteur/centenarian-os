## What This Course Covers

Coach & Gems is CentenarianOS's AI conversation module. Gems are custom AI personas you define — each one has a name, a purpose, and a system prompt that shapes how it thinks and responds. The Coach is where you talk to them. This course covers creating effective Gems, running coaching sessions, managing conversation history, and using the built-in flashcard generation feature.

## Key Takeaways

- Gems are custom AI personas; Coach is the chat interface that uses them
- Powered by Gemini 2.5 Flash — full multi-turn context per session
- System prompt defines the AI's behavior, expertise, and constraints
- Flashcard generation is a built-in feature activated by syntax in the system prompt
- Sessions are saved — conversations can be resumed at any time
