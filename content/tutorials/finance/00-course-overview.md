# Course Overview: Mastering Finance

**Course:** Mastering Finance
**Module:** Course Setup
**Duration:** ~2 min
**Lesson type:** text
**is_free_preview:** true
**CYOA navigation:** cyoa

---

## What This Course Covers

The Finance module is where you track income, expenses, and budgets alongside the rest of your life operating system. This course covers setting up budget categories, logging transactions, reading your dashboard, managing transaction history, and importing or exporting your data.

---

## Lessons in This Course

| # | Lesson | What you'll learn |
|---|--------|-------------------|
| 01 | Welcome to Finance | Dashboard overview, what the module tracks, how it connects to the rest of the system |
| 02 | Setting Up Budget Categories | Creating categories, assigning monthly budgets, color coding |
| 03 | Logging Transactions | Expense vs. income, all form fields, vendor, category assignment |
| 04 | Reading Your Dashboard | Summary cards, monthly trend chart, spending by category, budget progress bars |
| 05 | Transaction History | The transactions sub-page, filtering, searching, editing, and deleting |
| 06 | Importing and Exporting | CSV import for bulk data entry, CSV export for external tools |

---

## How to Navigate

This course uses **Choose Your Own Adventure** navigation. Lessons 01–03 are the recommended starting path — they cover the core setup and daily workflow. Lessons 04–06 are modular and can be visited in any order.

---

## Teacher Checklist

Before recording:

- [ ] At least 3 budget categories created with monthly budgets set
- [ ] At least 10 transactions logged across multiple categories (mix of income and expenses)
- [ ] At least 2 months of data so the monthly trend chart shows bars
- [ ] A sample CSV prepared for the import demo

---

## Key Takeaways

- Finance tracks income, expenses, and budgets in one module
- CYOA navigation — start with Lessons 01–03 for setup, then explore as needed
- No bank integrations — data is entered manually or imported via CSV
- Finance data contributes to the daily debrief financial snapshot and appears in the weekly review
