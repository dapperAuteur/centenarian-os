## Narrator Script

A focus session is a logged block of intentional work time. You start it, you work, you stop it, you rate how it went. That's the core loop. This lesson walks through every step of running a session in Simple mode — the standard timer.

---

### Starting a Session

From the Engine dashboard, click **Start a Focus Session**.

The session setup panel opens. Fill in the fields:

---

### Session Fields

**Activity** — What you're working on. Be specific: "Write intro section for podcast episode 3" is more useful than "writing." Specific entries make your session history searchable and your analytics meaningful.

**Milestone** (optional) — Link this session to a milestone in your Planner. When you link to a milestone, the session appears in that milestone's history, and the time can contribute to focus metrics for that goal.

**Task** (optional) — Link to a specific task under the milestone. This is the most granular link — it connects your deep work time directly to a scheduled task in your daily plan. If you link a task and mark it complete at session end, the task is checked off automatically.

**Duration** — How long you plan to work. The timer counts down from this time. Common choices: 25 min, 45 min, 60 min, 90 min. You can also set it to 0 for an open-ended session (timer counts up instead of down).

**Mode** — **Simple** for a single uninterrupted work block. **Pomodoro** for alternating work/break intervals. This lesson covers Simple mode. Pomodoro is covered in Lesson 03.

**Notes** (optional) — What you intend to accomplish, questions you want to answer, or context for this session. These show up when you review the session later.

---

### Running the Session

Click **Start**.

The timer begins. The UI shows:
- Time remaining (or elapsed, if open-ended)
- The activity name
- A pause button
- A stop/end button

Work on whatever you set out to work on. The timer runs in the background — you don't need to keep the tab focused.

If you get interrupted, click **Pause**. The timer freezes. Click **Resume** when you're ready to continue. The session logs actual focused time (paused time is excluded from the total).

When you're done (either the timer hits zero or you click **Stop**), the session ends and the completion screen appears.

---

### The Completion Screen

After stopping a session, you see the completion screen:

**Duration logged** — The actual time you worked (not counting paused time).

**Quality rating** — Rate this session from 1 to 5:
- **5** — Deep focus. In flow. Highly productive. No meaningful distractions.
- **4** — Good focus with minor interruptions. Solid output.
- **3** — Average. Some distraction. Got work done but not in the zone.
- **2** — Struggled to focus. Frequent distraction. Limited productive output.
- **1** — Couldn't focus. Mostly distracted. Very little accomplished.

This is subjective and that's fine. You're rating your experience, not your output. Sometimes a 3-quality session produces good work; sometimes a 5-quality session produces less than expected. Both data points are useful.

**Mark task complete** (if you linked a task) — Checkbox to check off the linked task immediately.

**Session notes** — Add any end-of-session notes: what you accomplished, what you didn't get to, what to continue in the next session.

Click **Save** to log the session.

---

### What Gets Logged

After saving, the session is recorded with:
- Activity name
- Date and time
- Duration (actual focused time)
- Quality rating
- Linked milestone and task (if set)
- Start and end notes

This data appears immediately in:
- Your Engine dashboard stats
- Your session history
- The weekly review (focus minutes and quality averages for the week)

---

### Practical Tips

**Don't overthink the setup.** A quick session with just an Activity and duration set is better than no session. The optional fields (milestone, task, notes) are for when they add value.

**Use the task link when you can.** Linking sessions to tasks in your Planner creates a clear connection between your planned work and your actual time on task. Over weeks, you can see which tasks took more time than estimated.

**Rate honestly.** A session where you got 20 minutes of real work in a 45-minute block is a 2 or 3, not a 5. Honest ratings make the analytics useful. Inflated ratings make the data noise.

## Key Takeaways

- Start a session from the Engine dashboard — set Activity, optional Milestone/Task link, Duration, and Mode
- Simple mode = single uninterrupted timer (counts down or up if duration = 0)
- Pause/Resume tracks actual focused time — paused time is excluded
- Quality rating (1–5) is the most important field — be honest, not optimistic
- Linking a task lets you check it off automatically at session end
- Session data appears immediately in dashboard stats and weekly review
