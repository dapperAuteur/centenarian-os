## Narrator Script

If you run the same type of session regularly — a morning writing block, a 25/5 Pomodoro for email, a 90-minute deep work session — setting up the same fields every time adds unnecessary friction. Session templates eliminate that friction. Define a session type once, then start it in one click.

---

### What Session Templates Are

A session template is a saved configuration for a focus session. It stores:
- Activity name (can be a general name like "Deep Work Block" or session-specific)
- Mode (Simple or Pomodoro)
- Duration (for Simple) or work interval / break interval / cycles (for Pomodoro)
- Default notes (optional) — a standing protocol or prompt you use for this session type

Templates don't store milestone or task links — those are set at session start because they vary each time. Everything else can be pre-filled.

---

### Creating a Template

From the Engine dashboard, click **Session Templates** (or navigate via the Engine menu).

Click **+ New Template**.

The template form opens — it's the same as the session setup form, but without the Milestone and Task fields. Fill in:

- **Template name** — What you call this template. This is what shows in your template list. "Morning Writing Block", "45-min Simple Session", "Email Pomodoros", "Deep Work 90".
- **Activity** — Pre-fill with a default activity name, or leave it editable so you can fill it in at session start.
- **Mode** — Simple or Pomodoro.
- **Duration** (Simple) or **Work/Break/Cycles** (Pomodoro).
- **Default notes** — Optional standing text for this session type. Useful for session protocols you follow: "Focus on one section only. No multitasking. Phone face-down."

Click **Save Template**.

---

### Using a Template

From the Engine dashboard, click **Start a Focus Session**. Above the manual session form, you'll see a **Templates** section with your saved templates listed as cards.

Click any template card to pre-fill the session form with that template's configuration. The Activity, Mode, Duration, and Notes fields fill in immediately.

You can then:
- Change the Activity if this session's specific focus differs from the template default
- Add a Milestone and Task link
- Adjust the duration or any other field
- Click Start

The template is a starting point, not a constraint. You can modify any field before starting.

---

### Editing and Deleting Templates

From the Session Templates list, each template has an **Edit** button and a **Delete** button.

Edit to update any field — name, mode, duration, default notes. Changes apply to future sessions started from this template; past sessions are unaffected.

Delete removes the template from the list. Past sessions that were started from this template are unaffected — sessions log the actual values used, not a reference to the template.

---

### How Many Templates Is Enough

Most users maintain 3–6 templates. Too few and you're still configuring everything manually. Too many and choosing a template adds its own cognitive load.

A useful template set for a typical user:
- **Morning Deep Work** — 90-min Simple, default notes with focus intention prompt
- **Standard Pomodoro** — 25/5/4 Pomodoro for administrative tasks
- **Quick Session** — 25-min Simple for short focused blocks
- **Writing Session** — 45-min Simple with default note: "One section, one draft, no editing"

Start with the session types you run most often and add more only if you find yourself reconfiguring the same setup repeatedly.

## Key Takeaways

- Templates save session configurations (mode, duration, activity, notes) for one-click session starts
- Templates don't store milestone/task links — those vary per session
- Create from Session Templates → + New Template; use by clicking a template card at session start
- All fields are pre-filled but editable — templates are a starting point, not a lock
- Keep the template list short (3–6) — enough to cover your regular session types without adding choice friction
- Deleting a template doesn't affect past sessions; editing only affects future ones
