## Narrator Script

Individual courses teach specific skills. Learning Paths are curated sequences of multiple courses designed to take you from where you are to a defined outcome — like building a complete longevity protocol from scratch, or mastering the analytics layer of CentenarianOS. This lesson covers how to find, follow, and track progress through a learning path.

---

### What Are Learning Paths?

A learning path is a named sequence of courses — usually 3 to 8 — grouped around a specific goal. They're created by teachers or the platform, and they tell you: "If you want to achieve X, take these courses in this order."

Examples:
- **CentenarianOS Foundation** — covers all platform modules in the recommended order
- **Sleep & Energy Protocol** — courses on health metrics, correlations, and fuel that focus on sleep optimization
- **High-Performance Athlete Track** — body tracking, engine analytics, travel, and recovery courses

---

### Finding Learning Paths

Navigate to `/academy/paths`. This is the Learning Paths catalog. Each path card shows:
- Path name and description
- Number of courses in the path
- Estimated total duration
- A "Recommended" badge if the platform suggests it based on your subscription tier or usage
- Your enrollment status (Not started / In Progress / Completed)

---

### Path Detail Page

Click any path card to open the path detail page at `/academy/paths/[pathId]`. Here you'll see:

**Path overview** — the goal of the path, who it's designed for, and what you'll be able to do after completing it.

**Course sequence** — an ordered list of courses in the path. Each course shows:
- Course title and teacher
- Short description of how it fits the path's overall goal
- Your enrollment and completion status for that specific course
- An **Enroll** or **Continue** button

**Your path progress** — at the top of the sequence: a progress bar showing how many courses in the path you've completed.

---

### Following a Path

Learning paths don't require separate enrollment — they're organizational structures that reference courses you may or may not already be enrolled in.

When you click **Enroll** on a course within a path, that's a standard course enrollment (same as enrolling from the catalog). The path tracks your progress across those courses automatically.

The recommended workflow:
1. Open the path detail page
2. Start with Course 1 — click Enroll (if needed) and then Go to Course
3. Complete that course at your own pace
4. Return to the path detail page and start Course 2
5. Repeat through the sequence

There's no obligation to follow paths in strict order — you can skip ahead or come back. The path tracker reflects reality: only courses you've completed are marked done.

---

### Path Progress Tracking

Your path progress bar shows:

```
path progress = courses completed in this path / total courses in path
```

A course counts as "completed" in the path when all its lessons are marked done (same definition as My Courses).

Completing all courses in a path marks the path as completed and adds it to a "Completed Paths" section on the My Courses page.

---

### Recommended Paths

The platform may show a **Recommended** badge on paths that align with your subscription tier or the modules you've used most. These are suggestions, not requirements.

If you're new to CentenarianOS and unsure where to start, look for the **CentenarianOS Foundation** path — it's designed to walk through all the platform's core modules in the sequence that builds best.

## Key Takeaways

- Learning Paths are curated multi-course sequences grouped around a defined outcome
- Find them at /academy/paths — each path has a name, description, course count, and duration
- Path detail page shows the ordered course sequence and your per-course status
- No separate path enrollment — enroll in each course within the path as you reach it
- Path progress = courses completed in path / total courses; all lessons done = course complete
- Completed paths appear in a "Completed Paths" section on My Courses
- New to CentenarianOS? Look for the CentenarianOS Foundation path to start
