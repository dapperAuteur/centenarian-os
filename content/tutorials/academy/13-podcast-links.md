## Narrator Script

Some lessons link to external podcast episodes on platforms like Spotify, Apple Podcasts, and YouTube. This lesson explains how podcast links work and how to use them alongside lesson content.

---

### Podcast Link Buttons

When a lesson includes podcast links, you'll see colored buttons at the top of the lesson content area, before the main text or video. Each button is labeled with the platform name:

- **Spotify** — green button with Spotify icon
- **Apple Podcasts** — purple button with Apple Podcasts icon
- **YouTube** — red button with YouTube icon
- **Amazon Music** — teal button
- **Other platforms** — gray button with a generic link icon

Click any button to open the episode on that platform in a new browser tab.

---

### Why Podcast Links?

Some courses are built around podcast episodes — the lesson content provides study guides, discussion questions, or reference material while the actual instruction happens in the audio episode.

This is common in courses like BVC (Better Vice Club) where each module corresponds to a podcast episode covering geography, economics, and social studies topics.

---

### Podcast Links + Chapter Markers

A powerful combination: a lesson can have both podcast links (for the external episode) and chapter markers (for a locally hosted audio version). The external links let you listen on your preferred platform, while the chapter-enabled audio player in the lesson gives you structured navigation within the Academy itself.

---

### Example Podcast Links Data

```json
[
  { "url": "https://open.spotify.com/episode/ABC123", "label": "Spotify" },
  { "url": "https://podcasts.apple.com/us/podcast/episode-name/id123", "label": "Apple Podcasts" },
  { "url": "https://www.youtube.com/watch?v=XYZ789", "label": "YouTube" }
]
```

The platform is auto-detected from the URL — teachers just paste the link and add a label.

---

### Not All Lessons Have Podcast Links

Podcast links are optional. If a lesson doesn't include them, the buttons won't appear. Most tutorial courses don't use podcast links — they're primarily for podcast-based curriculum courses.

## Key Takeaways

- Podcast links appear as colored buttons at the top of lesson content (Spotify, Apple, YouTube, etc.)
- Click any button to open the episode on that platform in a new tab
- Common in podcast-based courses where lessons pair audio episodes with study materials
- Can coexist with chapter-enabled audio players for structured in-app navigation
- Optional feature — only appears when the teacher adds podcast links to the lesson
