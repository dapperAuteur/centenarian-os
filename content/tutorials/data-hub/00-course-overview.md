## What You'll Learn

- How to navigate the centralized Data Hub
- Downloading and filling out CSV templates for any module
- Importing data via CSV file upload or Google Sheets published URL
- Exporting all data, date-range filtered data, and filtered datasets
- Module-specific import tips (vehicle nicknames, category auto-creation, milestone handling)
- Using Import/Export buttons on individual dashboard pages

---

## Key Takeaways

- The Data Hub at `/dashboard/data` is the central place for all data import/export
- Every module has a downloadable CSV template with example data
- Imports support CSV file upload and Google Sheets published URLs
- Exports support full download, date-range filtering, and module-specific filters
- Each dashboard page also has Import/Export buttons for quick access
- Max 1,000 rows per import (200 for vehicles, 500 for contacts)
